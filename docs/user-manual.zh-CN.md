# 用户手册

## 这套工作台能做什么

这个 MVP 一次只接受一个已结束的 Bilibili 录播/VOD 链接。

对每个任务，工作台可以：

- 下载源视频
- 生成中文字幕时间轴
- 生成中日双语字幕
- 生成可供人工复核的高光候选片段
- 在你确认候选片段后导出 MP4 切片
- 生成 Markdown 任务报告

这个 MVP 不支持直播录制、频道监控、订阅管理，也不会替你自动发布切片。

## 开始前准备

- 请使用**已结束的** Bilibili VOD 链接。
- 请从受信任的局域网设备访问 WebUI。
- 如果源视频是私有、会员限定或区域受限内容，请在提交任务前让操作人员准备好有效的 Bilibili Cookie 文件。

## 提交任务

1. 打开 WebUI，通常地址是 `http://<host>:5173`。
2. 在 **Task intake** 页面，把 Bilibili VOD 链接粘贴到 **Bilibili VOD URL** 输入框中。
3. 保持页面上可见的处理开关为开启状态。在当前 MVP 中，它们只起展示作用，不会改变后端流水线。
4. 点击 **Create task**。
5. 页面会直接跳转到 `/tasks/<task_id>`。

## 查看任务进度

任务详情页会按固定顺序展示标准阶段：

1. `ingest`
2. `media_prep`
3. `asr`
4. `translation`
5. `highlight`
6. `export`
7. `report`

你会看到：

- 任务运行中时，页面会较频繁刷新。
- 任务进入终态后，刷新频率会降低。
- **Stage timeline** 会显示当前阶段是运行中、已完成、已跳过还是失败。
- **Artifact overview** 会随着持久化产物写入而逐步补全。

如果任务失败，页面会展示导致任务停止的阶段和可读的失败摘要。在当前 MVP 中，需要由操作人员从后端/API 侧排查并发起重试。

## 复核字幕和高光候选

任务详情页下半部分就是工作区。

### 字幕复核

**Subtitle review and highlight confirmation** 区域会展示：

- 字幕段 ID
- 时间范围
- 中文字幕文本
- 双语字幕文本（如果翻译阶段已成功）

在导出切片前，你可以先在这里检查打轴和翻译质量。

### 高光复核

**Ranked candidate confirmation** 面板会展示：

- 排名
- 分数
- 原始片段时间范围
- 原因标签
- 可编辑的开始秒数和结束秒数

如果没有任何候选片段达到阈值，界面会明确显示“零候选”提示，而不是把结果隐藏起来。此时你仍然可以下载字幕和任务报告。

## 确认导出切片

在当前 MVP 中，切片导出是**手动确认**的。

1. 找到你想保留的高光候选片段。
2. 检查系统建议的时间范围。
3. 如有需要，调整 **Start (s)** 和 **End (s)**。
4. 点击 **Confirm export**。
5. 等待导出的切片卡片出现在页面中。

说明：

- 任务完成时系统不会自动导出切片。
- 导出结果是该确认区间对应的 MP4 文件。
- 如果时间范围无效，界面会显示后端返回的校验错误。

## 下载产物

**Artifact downloads** 区域可能会提供这些下载链接：

- 中文字幕
- 中日双语字幕
- 中文转写 JSON
- 双语转写 JSON
- 任务报告
- 已导出切片

**Exported clips** 区域会在切片导出成功后显示可下载的 MP4 文件。

常见情况：

- 如果你还没有确认任何候选片段，就不会看到导出的切片。
- 如果没有找到高光候选，字幕和任务报告仍然可能可用。

## MVP 范围边界

本手册只覆盖当前 MVP。

以下内容**不在当前版本中**：

- 直播录制
- 定时/周期任务
- 订阅
- 公网部署
- 多用户账号或审批流
